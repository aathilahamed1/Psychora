# **App Name**: MindWell Campus

## Core Features:

- AI-Guided Support Chat: An AI-powered chatbot offers instant, personalized coping strategies. This tool analyzes user input to provide appropriate advice and suggests professional referrals when necessary.
- Confidential Booking: A secure system for scheduling appointments with on-campus counselors or mental health helplines.
- Psychoeducational Resources: A curated library of videos, relaxation audio, and mental wellness guides. Serves static content, filtering according to user preference.
- Peer Support Platform: Moderated peer-to-peer support forum with trained student volunteers.
- Anonymous Data Analytics: An admin dashboard providing insights into student mental health trends using anonymous data. Helps authorities plan effective interventions.
- Language Selection: Allows users to switch between English and Hindi.
- Warm Hand-off System: A proactive system that bridges the gap between the digital app and the on-campus counselors. AI identifies students who may be at high risk and, with consent, alerts counselors anonymously, initiating a secure chat.

## Style Guidelines:

- Primary color: Soft lavender (#D1B0D7) to promote calmness and well-being.
- Background color: Light grey (#F0F0F0) for a clean and neutral backdrop.
- Accent color: Muted teal (#7FB5B5) to add a touch of serenity and trustworthiness.
- Body and headline font: 'PT Sans' (sans-serif) for a modern, readable, and welcoming feel.
- Use simple, friendly icons in a line art style to represent different mental health topics and resources.
- Maintain a clean, spacious layout with clear visual hierarchy to ensure ease of navigation and reduce cognitive load.
- Employ subtle, gentle animations for transitions and feedback to enhance user engagement without being distracting.
